# Session Changelog — 2026-03-04

## lins_ccc.py (new script)

- Created `scripts/lins_ccc.py` to compute Lin's Concordance Correlation Coefficient between Dektronics and Flatbed Net OD
- Computes CCC, Pearson r, and bias correction factor Cb
- 95% CI via bootstrap (5000 samples) — preferred for small n
- Outputs concordance plot with perfect concordance line (y=x), OLS regression line, and SD error bars per dose point
- Saves to `outputs/lins_ccc.png`
- Run: `python scripts/lins_ccc.py`

## drift_analysis.py (new script)

- Created `scripts/drift_analysis.py` to run drift/stability analysis on any normalized OD CSV
- Accepts file path as CLI argument — works on any dose level
- Tests: linear regression slope (is drift significant?), Mann-Kendall trend test, slope comparison between trials, paired t-test + Wilcoxon session offset, Grubbs outlier test
- Saves plot to `outputs/drift_analysis_<filename>.png`
- Run: `python scripts/drift_analysis.py data/normalized_data_0cGy.csv`

## testing.ipynb (updated)

- Added `plot_normalized_od(file_path, title)` function: scatter plot of normalized OD vs time with per-trial linear fit lines
- Prints slope, R², and p-value to console; keeps graph clean
- Works on any normalized data CSV with arbitrary number of trial columns

## lins_ccc_analysis.ipynb (new notebook)

- Interactive Jupyter notebook version of Lin's CCC analysis
- Widgets: dose point checkboxes, CI method toggle (Fisher z vs bootstrap), SD error bar and OLS line toggles
- Requires `ipywidgets`
