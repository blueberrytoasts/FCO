# Session Changelog — 2026-02-25

## plot_normalized_od.py (new script)

- Created `scripts/plot_normalized_od.py` to plot normalized OD vs time from any CSV
- CSV format: first column = time, remaining columns = OD readings (one per session/day)
- Handles different-length columns via `dropna()` (e.g. one series goes to 180 min, another to 120 min)
- Changed from line plot to scatter plot (no connecting lines)
- Default CSV path set to `data/normalized_data_0cGy.csv`
- Output saves to `outputs/normalized_od_<dose>.png`, where dose label is extracted from the filename (e.g. `normalized_data_0cGy.csv` → `0cGy`)
- Title updates automatically to include dose label
- Run: `python scripts/plot_normalized_od.py "data/your_file.csv"`
